Welcome To The Home Page For The SimpleChart Package.

Demo for implimenting Charts in WM Integration Server.


This sample package is created by Igor Androsov for purpose of demonstrating the creation of chart images from Integration Server. This package includes a simple chart creation service to build bar, line and area charts. These examples only a subset of charts that are available with JFFreeChart Java package. The chart generation code is based on Open Source free Java chart package JFreeChart Version 1.0.1, more information on this package and latest updates can be obtained at website: www.jfree.org/jfreechart. SimpleChart package includes a copy of JFreeChart in form of jar files distributed with IS package.

NOTE: If you are runing this package on Integration Server deployed on UNIX and do not have X11 or Display environment set, in this case additional configurations will be required to display charts graphics. User must iether configure UNIX Display environment settings or configure Integration Server JVM to use alternative methods to generate chart images. Use JVM 1.4 parameter: -Djava.awt.headless=true to generate chart images in UNIX/Linux system.
